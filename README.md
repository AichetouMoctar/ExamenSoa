Description

State Civil Service est une application développée en Spring Boot dans le cadre d’un projet SOA (Service-Oriented Architecture).

Le système permet la gestion :

 Personnes

 Mariages

 Actes de naissance

L’application utilise :

 REST API (Spring Boot)

 gRPC (communication interne – vérification d’identité)

 PostgreSQL (base de données)

Event Driven Architecture (publication d’événements)

 Spring Retry (résilience)

 Mock gRPC Server (pour test local)
